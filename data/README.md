# Data Files

The dataset is synthetic, deterministic, and generated by `scripts/generate_dataset.py`.

- `sales_history.csv`: 210 days of hourly covers with weather, weekend flag, event flag, sales, and top dish demand.
- `menu_recipes.csv`: recipe ingredient quantities per dish.
- `ingredients.csv`: current stock, unit, shelf life, supplier lead time, and minimum order quantity.
- `staff_rules.csv`: role-based staffing thresholds.
- `manager_corrections.csv`: sample manager feedback records.

Regenerate all files with:

```bash
python scripts/generate_dataset.py
```
